SPC2IT
======

Convert SPC files to IT (Impulse Tracker) files.

Compiling
=========

You need cmake.

> mkdir b && cd b && cmake .. && make

Installing
==========

FreeBSD
-------
> pkg install spc2it

Running
=======

Enter the executable's path in the command line, then press enter to see the syntax.
